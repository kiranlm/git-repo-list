### Description

A sample app for listing a user's git repos, created using React.js, webpack5 and typescript.

### Usage

##### Install

```bash
## clone the repo
git clone https://github.com/kiranlm/git-repo-list.git

## go to the directory
cd git-repo-list

### Available commands to run:

## install
npm install

## Run locally
npm run dev

## Production build
npm run build

## Serve production build
npm start

## Lint
npm run lint

## Run tests
npm run test

```
